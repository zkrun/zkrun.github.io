makefile在工程很多时可以用自动编译将效率提高很多

在用makefile之前需要安装make,本人是ubuntu系统所以直接

sudo apt-get install make

Makefile的本质是一个文件，需要配合make命令进行自动化编译。文件名常首字母大写。
在Makefile中的格式为:
<目标> : <依赖>
一个TAB<命令>
